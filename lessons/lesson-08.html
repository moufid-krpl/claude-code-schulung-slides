das problem
Jede Session startet bei Null — außer du gibst Claude ein Gedächtnis.
Ein Ordner mit Markdown-Dateien genügt: Claude liest ihn beim Start — und schreibt Neues selbst hinein.

lesson 08 · die struktur
Ein Index — dahinter Themen-Dateien
▸memory/ordnerstruktur
memory/  MEMORY.md              # Index: was liegt wo?  arbeitsweisen.md       # wie du arbeitest & entscheidest  projekt-wissen.md      # was im Code nicht steht  entscheidungen.md      # was entschieden wurde — und warum
Der Index bleibt kurz — die Details wohnen in den Themen-Dateien.
lesson 08 · anbindung
CLAUDE.md macht das Memory verbindlich
▸CLAUDE.mdausschnitt
## Memory- Liegt in: C:/claude-zentrale/memory/- Lies MEMORY.md zu Beginn jeder Session.- Neue dauerhafte Erkenntnisse: dort ergänzen.- Veraltetes dabei gleich entfernen.
Lesen beim Start, ergänzen bei neuen Erkenntnissen — beides steht als Regel in CLAUDE.md.
lesson 08 · inhalt
Was rein gehört — und was nicht

✓  Gehört rein

  • Dauerhafte Arbeitsweisen — „Deployments immer erst auf Staging"
  • Projektwissen, das nicht im Code steht — Kontext, Ansprechpartner, Eigenheiten
  • Entscheidungen + Begründung — damit sie niemand versehentlich rückgängig macht

✕  Bleibt draußen

  • Secrets & Zugangsdaten — die gehören in die .env (Lesson 07)
  • Tagesgeschäft — Einmal-Aufgaben veralten schneller, als du sie löschst
  • Was Git & Code schon wissen — Historie und Code dokumentieren sich selbst

lesson 08 · pflege
Ein Gedächtnis braucht Pflege
  • Nach großen Aufgaben aktualisieren lassen.„Was davon gehört ins Memory?" — Claude ergänzt selbst.
  • Veraltetes löschen.Ein falsches Gedächtnis ist schlimmer als gar keins.
  • Kompakt halten.Kurzer Index, knappe Themen-Dateien — keine Romane.
  • Ab und zu selbst reinlesen.Es ist Klartext in Markdown — dein Wissen, keine Blackbox.
lesson 08 · recap
Das nimmst du mit
  • Ohne Memory startet jede Session bei Null
  • Markdown-Ordner + Index — angebunden über CLAUDE.md
  • Rein: Arbeitsweisen, Wissen, Entscheidungen — keine Secrets
  • Pflege gehört dazu: ergänzen, ausmisten, kompakt halten
